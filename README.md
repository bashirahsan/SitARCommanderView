# SitARCommanderView
Repository for SitAR commander's View

#Description

This is the code for commanders view of the application. The commander's view is where the whole battlefield is put into surveillance by projecting the field using 3D tiles from mapbox on a flat surface.
The targets and track history are represented as 3D models on the top of 3D plot.

The extensive report on SitAR can be found at: https://drive.google.com/file/d/1d3qk4vAKc1HYPAwwyvljqCA0MH9mlLoR/view?usp=sharing

The report includes the following artifacts,

1. Literature Survey
2. System Architecture
3. Work Flow
4. Implementation Details
